# Customer Segmentation using K-Means Clustering

This project applies unsupervised machine learning techniques to group customers based on purchasing behavior.

## Project Overview
Customer segmentation helps businesses understand different types of customers and design targeted marketing strategies.

## Features
- Data preprocessing
- Exploratory Data Analysis
- K-Means clustering
- Customer group visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Workflow
1. Data analysis
2. Feature selection
3. Applying K-Means clustering
4. Determining optimal clusters
5. Visualizing customer segments

## Applications
- Targeted marketing
- Customer behavior analysis
- Business strategy optimization
